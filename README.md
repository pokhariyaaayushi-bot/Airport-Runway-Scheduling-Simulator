🛫 Airport Runway Scheduling Simulator

A Data Structures and Algorithms based Airport Traffic Management System designed to simulate runway scheduling, flight prioritization, and airport control operations.

The project demonstrates how real-world airport operations can be managed efficiently using Binary Search Trees (BST), Queues, and Priority Queues.

📌 Features
Flight Management
Add flights manually
Generate random flights automatically
Search flights using Flight ID
Cancel pending flights
Track flight status
Runway Scheduling
Landing and Takeoff request handling
Runway occupancy simulation
First-Come-First-Serve scheduling for normal flights
Automatic processing workflow
Emergency Handling
Emergency flight prioritization
Multiple emergency categories:
Engine Failure
Medical Emergency
Fuel Low
Priority-based runway allocation
Data Structures Used
Binary Search Tree (BST) for flight records
Queue for normal flight scheduling
Priority Queue for emergency flights
Additional Features
Flight waiting time calculation
Flight history management
Auto-cancellation of overdue flights
JSON-based data persistence
Interactive Graphical User Interface (GUI)
🛠 Technologies Used
C Programming Language
Python
Tkinter GUI Framework
JSON Storage
Data Structures & Algorithms
